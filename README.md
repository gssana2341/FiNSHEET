# 🚀 FiNSHEET — Smart Study & Note-taking Platform

[![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)](https://reactjs.org/)
[![Vite](https://img.shields.io/badge/Vite-646CFF?style=for-the-badge&logo=vite&logoColor=white)](https://vitejs.dev/)
[![Lucide](https://img.shields.io/badge/Lucide-FF4B4B?style=for-the-badge&logo=lucide&logoColor=white)](https://lucide.dev/)

**FiNSHEET** คือแพลตฟอร์มการเรียนรู้และจดบันทึกอัจฉริยะที่ออกแบบมาเพื่อเปลี่ยนประสบการณ์การเรียนของคุณให้ก้าวไปอีกขั้น ด้วยระบบจัดการเอกสารที่ทรงพลัง และเครื่องมือจดบันทึกระดับมืออาชีพที่ใช้งานง่ายเหมือนบน iPad

---

## ✨ Key Features (คุณสมบัติเด่น)

### ✍️ Premium Note Editor (Pro Tablet Interface)
สัมผัสประสบการณ์การจดบันทึกที่ลื่นไหลด้วยโหมด Editor เต็มรูปแบบ:
- **Unified Single TopBar**: แถบเครื่องมือเดียวที่รวบรวมทุกฟังก์ชันไว้ในที่เดียว เพื่อพื้นที่การจดบันทึกสูงสุด
- **Draggable Pro Palette**: แถบเลือกสีและขนาดเส้นที่สามารถลากย้ายตำแหน่งได้อิสระ พร้อมระบบดีดกลับขอบจอ (Magnetic Snapping)
- **Multi-page Navigation**: ระบบจัดการหลายหน้ากระดาษที่ลื่นไหล พร้อมแผงควบคุมหน้ากระดาษแบบลอยตัว

### 🤖 AI Study Buddy
ฟีเจอร์ผู้ช่วยอัจฉริยะที่พร้อมช่วยเหลือคุณในทุกหน้ากระดาษ:
- สรุปบทเรียนอัตโนมัติจากบันทึกของคุณ
- สร้าง Quiz และ Flashcards จากเนื้อหาในหน้ากระดาษที่คุณกำลังจด
- ปรึกษาข้อสงสัยเกี่ยวกับเนื้อหาการเรียนได้ตลอดเวลา

### 📦 Smart Library Management
จัดเก็บและจัดการไฟล์ของคุณอย่างเป็นระเบียบ:
- **3-Dots Action Menu**: ระบบเมนูที่จัดการทุกไฟล์ได้ง่ายๆ ทั้งการลบ, การสร้างควิซ หรือการเข้าสู่โหมดแก้ไข
- **Visual Badges**: สัญลักษณ์แจ้งสถานะไฟล์ชัดเจน (AI, Purchased, Free)

---

## 🛠️ Tech Stack (เทคโนโลยีที่ใช้)

- **Frontend**: [React](https://reactjs.org/) - พัฒนา UI ที่ลื่นไหลและตอบสนองไว
- **Build Tool**: [Vite](https://vitejs.dev/) - เพื่อการพัฒนาที่รวดเร็ว
- **Styling**: [Modern CSS & Glassmorphism](https://web.dev/glassmorphism/) - ดีไซน์ที่สวยงาม ทันสมัย และเบาสบายตา
- **Icons**: [Lucide React](https://lucide.dev/) - ชุดไอคอนที่เป็นมาตรฐานสากล

## 🚀 Getting Started (วิธีการเปิดใช้งาน)

1. **Clone the repository**
   ```bash
   git clone https://github.com/gssana2341/FiNSHEET.git
   ```

2. **Navigate into the directory**
   ```bash
   cd FiNSHEET
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start Development Server**
   ```bash
   npm run dev
   ```

เปิดบราวเซอร์ไปที่ `http://localhost:5173` เพื่อสัมผัสประสบการณ์ FiNSHEET!
